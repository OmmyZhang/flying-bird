Flying Bird
===

一个按住屏幕控制飞行方向的web小游戏。主要灵感来自Flappy Bird和Alto's Adventure。

纯Rust项目，使用Yew实现。

音乐:

- Fantasy world by Alex-Productions | Creative Commons CC BY 3.0
- String Ending by mriedl | Creative Commons 0
